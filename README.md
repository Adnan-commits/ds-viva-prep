# Data Science — Viva Preparation Resource

A free, open academic resource for students preparing for Data Science viva examinations.  
70 questions across 6 modules and lab experiments, grounded in prescribed textbooks.

---

## Live Site

🔗 https://adnan-commits.github.io/ds-viva-prep/

---

## What's Inside

- **70 viva-style questions** written as concise, spoken-register answers
- **7 sections** — 6 course modules + lab experiments
- **Textbook-grounded** — every answer cites its source
- **Tap to reveal** — answers expand on click, no page reloads
- **Progress tracker** — tracks how many answers you've opened in a session
- **Module filter** — jump directly to any module
- **Fully responsive** — works on mobile and desktop
- **No login, no ads, no tracking** — just the content

## How to Use

1. Open the live site link above
2. Use the module filter bar to jump to the topic you are revising
3. Tap any question to reveal the viva-ready answer
4. The progress bar at the top tracks how many answers you have opened
